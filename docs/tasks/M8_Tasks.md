# M8 タスク一覧

- 目的・背景は `docs/milestone/M8_PLAN.md` を参照。

## M8-A: ドキュメントとリリース手順

- [✅] `docs/dev/RELEASE_CHECKLIST.md` に「`gh release create` では exe を**必ず**パスで渡す／アップロードする」旨を明記し、コピペ用例を追加する。
- [✅] `.gitignore` の `build/` 例外を読みやすく整理し、意図（`nuitka` 配下は原則無視、exe のみ追跡可）が伝わるようにする。
- [✅] （任意）`gh release create ... build\nuitka\pdfDiffChecker.exe` をラップするスクリプトを `scripts/` に追加する。

## M8-B: コンボボックスと矢印キー

- [✅] `KeyboardNavigationShell` から **Down キーでの `ttk::combobox::Post` 自動実行**を削除する。
- [✅] **readonly コンボ**は **Enter / KP_Enter** のみで Post する（既存ハンドラを維持・整理）。
- [✅] ポップダウンが**閉じている**ときは、コンボ上の **Up/Down を空間フォーカス**の対象に含める。ポップダウンが**開いている**ときは従来どおりリストが矢印を処理する。

## M8-C: 差分強調（レガシー統一ロジックへ戻す）

- [✅] `build_diff_highlight_overlay_rgba` を **M6 初期の画素差分＋開閉・膨張**のみに戻す（インク XOR／same-cell 補助パラメータは削除）。
- [✅] `main_tab` の差分オーバーレイは **二色化／指定色濃淡で分岐しない**（処理済みフル解像度ラスタのみ使用）。パレット色は平均して単一 `highlight_rgba` に反映。

## M8-D: プレビュー進捗と変身後のページ位置

- [✅] **ページ遷移**（前へ／次へ／ページ番号入力）でプレビュー再描画中にプログレスバー（`U182`）。
- [✅] **変身ボタン**で変換完了後、プレビュー再描画中にプログレスバー（`U183`）。PDF→PNG 変換自体は既存の `ProgressWindow` を継続利用。
- [✅] **二値化／指定色濃淡**プルダウン変更時にプログレスバー（`U184`）。
- [✅] **差分を強調表示**チェック切替時にプログレスバー（`U185`）。
- [✅] 変身（DPI 反映）後の `current_page_index` を、変身**前**のインデックスに**クランプして維持**（`page_count` 減少時は末尾に合わせる）。
